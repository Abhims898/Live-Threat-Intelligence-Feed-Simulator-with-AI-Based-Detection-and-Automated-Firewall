🛡️ Cyber Threat Intelligence Dashboard

📌 Overview

This project simulates a Cyber Range environment using real-time threat intelligence and AI-based anomaly detection.

🚀 Features

- Real-time threat detection
- AbuseIPDB API integration
- Automated firewall blocking
- AI-based anomaly detection
- SOC-style dashboard
- Graph visualization

🛠️ Technologies

- Python (Flask)
- Chart.js
- Scikit-learn
- Kali Linux

▶️ How to Run

- git clone Git hub link: https://github.com/Abhims898/Live-Threat-Intelligence-Feed-Simulator-with-AI-Based-Detection-and-Automated-Firewall.git
- cd cyber_dashboard
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- AI&API_Threat detection.py
